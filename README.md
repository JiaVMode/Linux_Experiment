# Linux 系统信息查询工具

本项目包含两个Linux实验：C语言命令行交互程序和Shell脚本交互程序，用于查询Linux系统信息。

## 项目结构

```
├── exp1/                    # 实验一：C语言程序
│   ├── Makefile            # 项目构建文件
│   ├── include/            # 头文件目录
│   │   ├── colors.h       # ANSI颜色定义
│   │   ├── menu.h         # 菜单模块
│   │   ├── logger.h       # 日志模块
│   │   ├── cpu_info.h     # CPU信息
│   │   ├── process_info.h # 进程信息
│   │   ├── memory_info.h  # 内存信息
│   │   └── storage_info.h # 存储设备信息
│   └── src/                # 源文件目录
│       ├── main.c
│       ├── menu.c
│       ├── logger.c
│       ├── cpu_info.c
│       ├── process_info.c
│       ├── memory_info.c
│       └── storage_info.c
│
└── exp2/                    # 实验二：Shell脚本
    └── menu.sh             # 交互式菜单脚本
```

## 实验一：C语言命令行交互菜单

### 功能
- A: 查看CPU硬件信息
- B: 查看进程运行信息
- C: 查看内存硬件信息、内存使用信息
- D: 查看存储设备硬件信息、存储设备使用信息
- E: 退出

### 特性
- 模块化设计，多文件组织
- Makefile管理编译
- 彩色终端输出
- 日志记录到log.txt
- 支持GDB调试

### 编译运行
```bash
cd exp1
make          # 编译
./sysinfo     # 运行
make debug    # GDB调试
make clean    # 清理
```

## 实验二：Shell脚本交互菜单

### 功能
- A: 查看进程信息
- B: 查看用户信息
- C: 显示当前目录信息
- D: 显示计算机名、Linux内核版本、当前IP地址
- E: 退出

### 特性
- 显示系统日期（年/月/日格式）
- 彩色终端输出
- 模块化函数设计

### 运行
```bash
cd exp2
chmod +x menu.sh
./menu.sh
```

## 技术说明

- **系统信息获取**：读取/proc文件系统 + popen执行shell命令
- **彩色输出**：使用ANSI转义码
- **磁盘使用率着色**：<70%绿色, 70-90%黄色, >90%红色

## 环境要求

- Linux操作系统
- GCC编译器
- Bash Shell
- GDB调试器（可选）
